Our program is a router that takes the details that specifies the size of the grid, the obstacles and lists the nets to be routed. 
And, it produces the output in a text file that conatins all the cells used by each net. 

How to use the program:

	1- Change all the file paths to yours. For example, change the input and the output file paths. 
	2- Run the C++ code in visual studio. 
	3- Run the python script to see the nets visualized on the screen.

Be Careful to follow the same input syntax otherwise the program will not function proberly. 
Also, pay attention to the indices of the source and targets; make sure it is not out of range.
